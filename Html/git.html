Git

  • git init - инициализация нового репозитория
  • git clone [url] - клонирование удаленного репозитория
  • git status - проверка состояния файлов в рабочей директории
  • git add [файл] - добавление файлов в индекс (staging area)
  • git commit -m "сообщение" - фиксация изменений с комментарием
  • git push - отправка изменений в удаленный репозиторий
  • git pull - получение изменений из удаленного репозитория
  • git branch - работа с ветками (просмотр, создание, удаление)
